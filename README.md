## Installation

```
npx tailwindcss init
```

Add this in CSS Input File

```
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Configuration

```
// update config file [content]
```

```
For adding CSS Classes
npx tailwind -i [css input file] -o [css output file] --watch
```

```
npx tailwind -i ./input.css -o ./style.css --watch
```
